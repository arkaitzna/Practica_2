Ejercicio 12 y 13

Ejercicio 12:



Para crear un archivo TypeSript, simplemente creamos un archivo en nuestro editor con la extensión .ts --> index.ts .Podemos darle un contenido como un console.log... A continuación debemos asegurarnos de que tenemos typescript con 'tsc -v' y si no lo tenemos instalarlo con 'npm install -g typescript'. A continuación transpilariamos el archivo desde la consola con el comando 'tsc index.ts' y nos creara un archivo llamado 'index.js' el cual seria la transpilación de index.ts.


Ejercicio 13:



Para saber las versiones que tienes instaladas de TypeScript, Node.js, el instalador de paquetes y Angular, podemos utilizar los siguientes comandos en la consola:
- Para TS: tsc -v
- Para Node: node -v
- Para instalador de paquetes: npm -v
- Para Angular: ng version

Aprendamos Front-End

Bienvenidos a nuestra página de turotiales de programación web enfocados en el frontend! Aquí encontrarás una gran variedad de contenidos para mejorar tus habilidades en diseño y desarrollo web.

Introducción a HTML y CSS:
Aprende los fundamentos de HTML y CSS para construir páginas web estáticas y dinámicas. Aprenderás a crear una estructura básica de HTML y a darle estilo con CSS.
Diseño responsive:
Aprende a crear diseños adaptables a distintos tamaños de pantalla. Descubre cómo utilizar media queries, flexbox y grid para crear diseños responsivos que se adapten a cualquier dispositivo.
Animaciones CSS:
Aprende a crear animaciones CSS para darle vida a tus diseños. Conoce las distintas propiedades y técnicas para crear animaciones suaves y atractivas. Con ellas podras darle estilo a tu web.
Diseño web con Bootstrap:
Utiliza el framework Bootstrap para crear diseños web rápidamente. Descubre cómo utilizar sus componentes y estilos predefinidos para crear sitios web responsivos y con un aspecto moderno
Optimización de rendimiento:
Aprende a optimizar el rendimiento de tus sitios web. Conoce técnicas para reducir el tiempo de carga de tus páginas y mejorar la experiencia de usuario. Tu web irá mas rapido
JavaScript con frameworks:
Aprende los fundamentos de JavaScript para añadir interactividad a tus sitios web. Descubre cómo utilizar variables, condicionales y bucles para crear scripts simples.
Accesibilidad web:
Crea sitios webs para personas con discapacidades. Descubre las pautas de accesibilidad y utiliza herramientas de validación para asegurarte de que tus sitios web sean accesibles para todos.
Optimización de SEO:
Aprende a optimizar el SEO de tus sitios web. Conoce técnicas para mejorar la visibilidad de tus páginas en los motores de búsqueda y atraer más tráfico a tu sitio web.
Herramientas de desarrollo:
Utiliza herramientas de desarrollo como Chrome DevTools para depurar y optimiza tus web. Conoce las distintas funcionalidades y técnicas para utilizar estas herramientas de manera efectiva.
Esperamos que estos contenidos te sean de ayuda para mejorar tus habilidades en programación web y diseño frontend. ¡Aprende con nosotros y conviértete en un experto en desarrollo web!
